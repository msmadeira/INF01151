# INF01151

## Building

At the root of the project, in a unix terminal run:

`make`

### Server only

`make server`

### Client only

`make client`

## Running

### Server

`make runser`

Or, after building:

`./server`

### Client

`make runcli1`
`make runcli2`
`make runcli3`

## Cleaning

`make clean`
